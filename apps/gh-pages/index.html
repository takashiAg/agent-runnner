agent-runnner
Workflow Package GitHub

GitHub-native AI runner

Issue から PR までを、安全な runner でつなぐ。

story / bug の整理、task 分解、worktree 上での自動修正、PdM・PjM・Tech Lead・Engineer 観点のレビューを、Node.js CLI と GitHub Actions で運用するための実装基盤です。

npm install -g label-triggered-ai-runner label-ai-runner --help

Workflow

Plan

Issue の label から story / bug / task を読み、実装前の方針と未決事項を整理します。

Split

monorepo の BE / FE / shared 境界を読み、1 PR でレビューしやすい task に分けます。

Run

Codex CLI / Claude Code CLI は provider として起動し、LLM にはコマンド実行権限を渡しません。

Review

PR に対して PdM、PjM、Tech Lead、Engineer の観点を分けてコメントします。

Package

このリポジトリは pnpm workspace の monorepo です。CLI は packages/runner、公式サイトは apps/gh-pages にあります。
